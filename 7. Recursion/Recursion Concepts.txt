


Here’s a comprehensive list of problems to master **Recursion**, covering all major topics and concepts:

---

### **1. Basic Recursion Problems**
1. **Print Numbers**:
   - Print numbers from 1 to \(n\) and \(n\) to 1 using recursion.
2. **Factorial of a Number**:
   - Compute the factorial of a given number \(n\).
3. **Fibonacci Sequence**:
   - Print the \(n\)-th Fibonacci number or the first \(n\) Fibonacci numbers.
4. **Sum of Digits**:
   - Calculate the sum of digits of a number.
5. **Reverse a String**:
   - Reverse a string using recursion.

---

### **2. Subset and Combination Problems**
1. **Subsets of a Set**:
   - Generate all subsets (power set) of a given set.
2. **String Permutations**:
   - Generate all permutations of a given string.
3. **Combinations**:
   - Find all combinations of size \(k\) from an array.
4. **Letter Case Permutation**:
   - Generate all letter case permutations of a string (e.g., "a1b2" → ["a1b2", "A1b2", "a1B2", "A1B2"]).

---

### **3. Array and Sequence Problems**
1. **Binary Search**:
   - Implement binary search using recursion.
2. **Find the Maximum/Minimum in an Array**:
   - Use recursion to find the maximum or minimum element in an array.
3. **Check if Array is Sorted**:
   - Determine if an array is sorted using recursion.
4. **Tower of Hanoi**:
   - Solve the Tower of Hanoi problem for \(n\) disks.

---

### **4. String Problems**
1. **Palindrome Check**:
   - Check if a string is a palindrome using recursion.
2. **Count Subsequences**:
   - Count all subsequences of a string that match a given pattern.
3. **Generate Parentheses**:
   - Generate all valid combinations of \(n\) pairs of parentheses.

---

### **5. Mathematical Problems**
1. **Power Function**:
   - Compute \(a^b\) using recursion.
2. **Greatest Common Divisor (GCD)**:
   - Find the GCD of two numbers using recursion.
3. **N Choose K (Binomial Coefficient)**:
   - Compute \(C(n, k)\) using recursion.
4. **Josephus Problem**:
   - Solve the Josephus problem for \(n\) people and a step size \(k\).

---

### **6. Backtracking Problems (Recursion + Decision Making)**
1. **N-Queens Problem**:
   - Place \(N\) queens on an \(N \times N\) chessboard.
2. **Sudoku Solver**:
   - Solve a partially filled Sudoku grid.
3. **Rat in a Maze**:
   - Find all paths for a rat to move from the top-left to the bottom-right of a maze.
4. **Word Search**:
   - Find if a word exists in a 2D grid.

---

### **7. Divide and Conquer Problems**
1. **Merge Sort**:
   - Implement merge sort using recursion.
2. **Quick Sort**:
   - Implement quick sort using recursion.
3. **Count Inversions in an Array**:
   - Count the number of inversions in an array using merge sort.

---

### **8. Advanced Problems**
1. **Kth Symbol in Grammar**:
   - Find the \(k\)-th symbol in a grammar sequence.
2. **Gray Code**:
   - Generate \(n\)-bit Gray codes using recursion.
3. **Expression Evaluation**:
   - Evaluate a mathematical expression using recursion.
4. **Split Array into Equal Sum Subsets**:
   - Check if an array can be split into two subsets with equal sums.

---

### **9. Tree Problems**
1. **Binary Tree Traversals**:
   - Preorder, Inorder, and Postorder traversals using recursion.
2. **Height of a Binary Tree**:
   - Compute the height of a binary tree.
3. **Lowest Common Ancestor**:
   - Find the lowest common ancestor of two nodes in a binary tree.
4. **Maximum Path Sum**:
   - Find the maximum path sum in a binary tree.

---

### **10. Dynamic Programming Problems Using Recursion**
1. **Climbing Stairs**:
   - Count ways to climb \(n\) stairs, taking 1 or 2 steps at a time.
2. **0/1 Knapsack**:
   - Solve the knapsack problem using recursion.
3. **Longest Common Subsequence (LCS)**:
   - Find the LCS of two strings using recursion.
4. **Coin Change Problem**:
   - Count the number of ways to make change for a given amount.

---
